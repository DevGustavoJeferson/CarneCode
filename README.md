# 🥩 CarneCode

Aplicação web desenvolvida como projeto de extensão para a faculdade.

O objetivo do projeto é facilitar a consulta de códigos de produtos de carnes, substituindo tabelas impressas por uma solução digital mais rápida e acessível.

- 🔗 Acesse o projeto: [Link do projeto](https://devgustavojeferson.github.io/CarneCode/)

---

## 📌 Sobre o Projeto

Em ambientes onde existem diversos cortes e categorias de produtos, a busca manual por códigos pode se tornar demorada e aumentar a possibilidade de erros.

O CarneCode foi criado para organizar essas informações em uma página web responsiva, permitindo que o usuário encontre produtos através de uma barra de pesquisa.

---

## 🎯 Objetivo

Criar uma ferramenta simples para:

- Facilitar a busca de códigos de produtos;
- Melhorar a produtividade durante o trabalho;
- Reduzir o uso de folhas impressas;
- Organizar informações de forma mais acessível.

---

## 🚀 Funcionalidades

- 📋 Tabelas organizadas por categorias de produtos;
- 🔎 Sistema de busca em tempo real;
- 💡 Sugestões conforme o usuário digita;
- ✨ Destaque visual do produto encontrado;
- 📱 Interface pensada para dispositivos móveis.
