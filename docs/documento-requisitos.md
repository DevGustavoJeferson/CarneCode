# CarneCode - Documento de Requisitos

## Descrição

Este projeto tem como ideia uma página web com foco mobile, contendo códigos de carnes que facilitarão a busca pelos produtos, promovendo maior eficiência no trabalho e evitando impressões desnecessárias de folhas A4.

## Problema que Resolve

Uma mesma carne pode ser cortada de diferentes maneiras e destinada a diferentes locais, fazendo com que o código varie de acordo com cada situação. Com tantas variações, o funcionário precisa procurar manualmente o código correto em diversas tabelas impressas em folhas A4, aumentando as chances de erros humanos.

A página web, além de facilitar o acesso aos códigos, contará com um sistema de busca para auxiliar na localização do código correto, aumentando a produtividade do trabalho e reduzindo o uso desnecessário de insumos.

## Funcionalidades

* Códigos organizados em tabelas, com um layout confortável e de fácil visualização.
* Sistema de busca de códigos por meio da descrição do produto.

## Fluxo de Uso

1. O usuário apontará a câmera do celular para um QR Code contendo o link da página.
2. Ao acessar a página, encontrará o campo de busca de códigos, onde poderá descrever o produto desejado. O sistema o direcionará para a localização correspondente na tabela.
3. O código encontrado receberá um destaque visual para facilitar sua identificação.
