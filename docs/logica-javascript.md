# Lógica Javascript

1. O usuário digita o nome do produto no campo de busca.
2. O sistema captura o texto digitado.
3. O sistema percorre as descrições dos produtos presentes nas tabelas HTML.
4. O sistema compara o texto digitado com os produtos, ignorando diferenças entre letras maiúsculas e minúsculas.
5. Enquanto o usuário digita, o sistema exibe sugestões de produtos encontrados.
6. Ao selecionar uma sugestão:
    - o sistema identifica o produto correspondente;
    - destaca a linha do produto dentro da tabela;
    - facilita a visualização do código.
7. Caso nenhum produto seja encontrado:
    - exibe a mensagem "Produto não encontrado".
